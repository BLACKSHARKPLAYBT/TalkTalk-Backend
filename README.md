# TalkTalk后端项目
## 项目简介
TalkTalk的后端接口，用于API调用。
数据库使用的是**mysql数据库**

|                 环境变量            |     说明   |
|:--------------------------------------:|:----------:|
|                    sql_user            | 用户名 |
|                  sql_password          |密码|
|                    sql_host            |地址|            
|                  sql_database          |默认数据库|           
|                    sql_port            |端口号|
